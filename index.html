Sobre o Projeto e o Jogo

Pentago é um jogo de tabuleiro abstrato para dois jogadores, inventado por Tomas Flodén. O objetivo é simples: ser o primeiro a conseguir cinco peças de sua cor em uma linha, seja na horizontal, vertical ou diagonal. O que torna o jogo único é que, após colocar uma peça, o jogador deve girar um dos quatro quadrantes do tabuleiro em 90 graus. Essa mecânica de rotação transforma um jogo aparentemente simples em um desafio estratégico complexo e dinâmico.

Nossa proposta é criar uma versão digital e funcional deste jogo utilizando tecnologias web modernas, como parte da avaliação da disciplina de Programação Web.

Objetivos do Projeto

Os principais objetivos deste trabalho são:

  • Aplicar os conceitos de HTML semântico para estruturar uma página web de forma clara e organizada.
  • Utilizar CSS para estilizar a página, criando um layout agradável e responsivo.
  • Aprofundar o conhecimento sobre a manipulação do DOM (Document Object Model) com JavaScript no futuro.
  • Desenvolver a lógica de programação necessária para implementar as regras do jogo Pentago.
  • Trabalhar em equipe utilizando ferramentas de versionamento de código como o Git e o GitHub.

A Disciplina

Este projeto está sendo desenvolvido no contexto da disciplina de Programação Web. O foco inicial é a construção da estrutura da página (HTML) e sua estilização (CSS), estabelecendo a base para as próximas fases que envolverão a implementação da lógica do jogo com JavaScript.
