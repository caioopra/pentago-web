Árvore do Document Object Model (DOM)

Abaixo está uma representação simplificada da hierarquia de elementos (árvore DOM) da nossa página index.html. 

html
|-- head
|   |-- meta (charset)
|   |-- meta (viewport)
|   |-- title
|   |-- link (favicon)
|   |-- link (stylesheet)
|
|-- body
    |-- header
    |   |-- h1
    |   |-- nav
    |       |-- ul
    |           |-- li -> a
    |           |-- li -> a
    |           |-- li -> a
    |           |-- li -> a
    |
    |-- div (class="container")
    |   |-- aside
    |   |   |-- h2
    |   |   |-- h3
    |   |   |-- ul -> li, li, ...
    |   |   |-- h3
    |   |   |-- p -> a
    |   |   |-- h3
    |   |   |-- p
    |   |   |-- h3
    |   |   |-- ul -> li, li
    |   |   |-- h3
    |   |   |-- a
    |   |
    |   |-- main
    |       |-- section (id="sobre")
    |       |   |-- h2
    |       |   |-- p
    |       |   |-- p
    |       |
    |       |-- section (id="objetivos")
    |       |   |-- h2
    |       |   |-- p
    |       |   |-- ul -> li, li, ...
    |       |
    |       |-- section (id="disciplina")

    |-- footer
